# CRISPR prediction tools

CRISPR 기반의 유전자 가위 효율을 예측하는 딥러닝 모델이 김형범 교수님 연구실에서 많이 개발되었다. 김형범 교수님 연구실 외에도 다른 그룹에서도 유사한 모델들을 개발해서 발표한 것들이 있다.

이를 이용하면 누구나 손 쉽게 고성능의 유전자 가위를 선정하고 연구에 활용할 수 있다.&#x20;
